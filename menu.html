專案列表 :

  • 美國文學報告

    因應用外語系大三下的美國文學課中的期末報告要求，須發揮創意，並展現所學，所以我做了一頁網頁來將我學到東西呈現在上面。這是我的第一個製作的網頁，在 HTML 與 CSS 運用上都顯得很生疏。

  • 六角學院前端體驗營 (履歷版型)

    因緣際會報名了這個免費的大學生體驗營，開始對網頁切版有基礎的認識，這是此營隊的第一項主要作業，使用六角學院給的 Adobe XD 版型，來進行切版。第一次了解到 CSS Reset 的使用方法。

  • 六角學院前端體驗營 (甜點電商版型)

    這是此營隊的第二項主要作業，使用六角學院給的Adobe XD版型，來進行切版。這次則是認識到 Flex, hover, 一些 HTML5 與 CSS3 的基本使用以及如何實現 RWD (響應式網頁設計)。

  • 六角學院前端體驗營 (香水電商版型)

    這是此營隊的第三項主要作業，使用六角學院給的Adobe XD版型，來進行切版。這次主要運用到了 BS4來進行排版以及一些 JQuery 和使用 SCSS 來修改幾項 BS4 預設值 。
